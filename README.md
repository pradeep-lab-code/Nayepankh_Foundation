# NayePankh Foundation Website

A responsive NGO website built using React.js and Tailwind CSS for NayePankh Foundation. The website aims to showcase the organization's mission, programs, impact, and volunteer opportunities while providing users with an easy way to get in touch.

## Features

* Responsive Design (Mobile, Tablet, Desktop)
* Modern UI with Tailwind CSS
* Smooth Scroll Navigation
* Framer Motion Animations
* Volunteer Registration Form
* Contact Form Integration using Web3Forms
* Multiple Informative Sections
* Clean and Accessible Layout

## Sections

* Home
* About Us
* Programs
* Impact
* Volunteer
* Contact
* Footer

## Tech Stack

* React.js
* Tailwind CSS
* Framer Motion
* Web3Forms
* Vite

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd nayepankh-foundation
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```

## Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_WEB3FORMS_KEY=your_web3forms_access_key
```
